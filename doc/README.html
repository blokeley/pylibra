Libra documentation

Libra is an application to read data from a serial port, parse it into useful chunks, and write the data to a file. The original version was in Java but this port was written in Python to reduce development time, and for fun!

User instructions

Developer instructions

To develop Libra you simply need a python interpreter and a couple of libraries:

  1. Python interpreter. For example, Python 2.6.
  2. pywin32 Python extensions for Windows.
  3. pySerial serial port library.
  4. py2exe to create the Windows executable.
  5. A good text editor. I use eclipse 3.4 with pyDev.
